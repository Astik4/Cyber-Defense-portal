# 🛡️ Cyber Defense Portal

An end-to-end SOC simulation platform focused on real-time network monitoring, threat detection, AI-powered analysis, and security event visualization.


## 🚀 Features

* 📡 Live packet sniffing and monitoring
* 🚨 Real-time threat alerts
* 🧠 AI-powered threat analysis
* 🔍 Deep system scanning
* 📊 Interactive SOC dashboard
* 💾 Alert logging using SQLite
* ⚡ Real-time updates with WebSockets


## 🛠️ Tech Stack

* Python
* Flask
* React
* Socket.IO
* Scapy
* SQLite
* CSS


## 📌 Modules

### 🔹 Packet Monitoring

Captures and analyzes network packets in real time to detect suspicious activity.

### 🔹 Threat Detection

Uses heuristic-based detection techniques to identify potential threats and abnormal behavior.

### 🔹 AI Threat Analyzer

Analyzes suspicious indicators and generates threat insights using AI-assisted analysis.

### 🔹 SOC Dashboard

Interactive dashboard for monitoring alerts, packets, scans, and system activity.

### 🔹 Alert Logging

Stores detected threats and alerts locally using SQLite.


## ⚙️ Installation

```bash
git clone https://github.com/Astik4/cyber-defense-portal.git

cd cyber-defense-portal

pip install -r requirements.txt

python backend/app.py
```
